I thought it would be cool to try and make an image/content slider using only HTML, and css. I was hoping to make it completely navigate-able (I think that's a word) by using the keyboard. FYI I am writing this as I am figuring it out, so there will probably be some trial and error here.

That said, my first step is to see how tab index and the keys work with a pretty plain set of radio buttons. 

<div class="radio-slider-container">
  <input type="radio" name="radio" value="" id="radio1">
  <label for="radio1">Label 1</label>
  <input type="radio" name="radio" value="" id="radio2">
  <label for="radio2">Label 2</label>
  <input type="radio" name="radio" value="" id="radio3">
  <label for="radio3">Label 3</label>
</div>

If you click the paragraph before the examples, and hit tab, you bring focus to the radio buttons. The right and left arrow keys then work to select different options. The up and down arrow keys work too. Below I added some images, and the ability to shuffle through the images with the radio buttons. Right now the code is very un re-usable and responsive, but that's for another time.

Again, it's not responsive or scalable (number of slides) yet but I at least got something working where I can build of the concept.
